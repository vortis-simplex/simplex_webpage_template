## Overview

The Notion integration brings AI-powered image generation directly into your Notion workspace. Create diagrams, illustrations, and visual content without leaving your favorite productivity tool.

## Key Features

- **Slash Commands**: Generate images with simple `/powerai` commands
- **Database Integration**: Auto-generate images based on database properties
- **Page Covers**: Create unique covers for your Notion pages
- **Template Support**: Use custom templates for consistent visuals
- **Collaborative Editing**: Share and edit AI-generated images with your team

## How It Works

1. **Connect Your Workspace**: Authorize PowerAI to access your Notion workspace
2. **Install the Extension**: Add the PowerAI extension to your browser
3. **Use Slash Commands**: Type `/powerai` in any page to generate images
4. **Customize Output**: Adjust style, size, and format preferences
5. **Insert and Share**: Add generated images to your pages and databases

## Use Cases

- **Documentation**: Create diagrams and illustrations for technical docs
- **Project Management**: Generate visual timelines and flowcharts
- **Knowledge Base**: Create custom icons and graphics for your wiki
- **Meeting Notes**: Generate visual summaries and mind maps
- **Personal Planning**: Create vision boards and goal visualizations

## Setup Instructions

To set up the Notion integration:

1. Go to PowerAI Integrations → Notion
2. Click "Connect" and authenticate with Notion
3. Grant permissions for page and database access
4. Install the PowerAI browser extension
5. Configure your default image settings
6. Test with a slash command in any Notion page

## Technical Details

- **API Version**: Notion API v2022-06-28
- **Authentication**: OAuth 2.0 with integration tokens
- **Supported Content**: Pages, databases, blocks
- **Image Formats**: PNG, JPG, WebP
- **Max Resolution**: Up to 2048x2048 pixels

## Slash Commands

- `/powerai generate [prompt]` - Generate an image from a prompt
- `/powerai cover [prompt]` - Set as page cover
- `/powerai icon [prompt]` - Set as page icon
- `/powerai diagram [type]` - Generate a diagram type

## Pricing

Notion integration is free to install. Image generation uses your PowerAI quota. Team plans include advanced features like database automation and custom templates.
