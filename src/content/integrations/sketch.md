## Overview

The Sketch integration brings AI-powered image generation directly into your Sketch design workflow. Create stunning visuals, illustrations, and graphics without leaving your favorite design tool.

## Key Features

- **Native Plugin**: Seamless integration as a Sketch plugin
- **Smart Layers**: Generate images directly into selected layers
- **Symbol Support**: Create AI-powered symbols for reuse
- **Style Preservation**: Maintain your design system's visual consistency
- **Version History**: Track and revert to previous AI generations

## How It Works

1. **Install the Plugin**: Download and install the PowerAI Sketch plugin
2. **Authenticate**: Connect your PowerAI account
3. **Select a Layer**: Choose any layer or artboard in your design
4. **Generate Content**: Use the plugin to generate AI images
5. **Refine and Export**: Edit and export your generated visuals

## Use Cases

- **UI Design**: Generate custom illustrations and icons for interfaces
- **App Mockups**: Create realistic screenshots and device mockups
- **Brand Assets**: Generate logos, patterns, and brand graphics
- **Marketing Materials**: Create banners, ads, and promotional graphics
- **Prototyping**: Generate placeholder images and content for prototypes

## Setup Instructions

To set up the Sketch integration:

1. Download the PowerAI plugin from the Sketch plugin marketplace
2. Install the plugin in Sketch (Plugins → Manage Plugins)
3. Open the plugin and sign in with your PowerAI account
4. Configure your preferred settings and style presets
5. Start generating images directly in your designs

## Technical Details

- **Plugin Version**: v1.8.0
- **Sketch Support**: Sketch 70 and above
- **API Integration**: RESTful API with real-time updates
- **Supported Formats**: PNG, JPG, SVG, PDF
- **Max Resolution**: Up to 8192x8192 pixels
- **Processing Time**: 3-15 seconds depending on complexity

## Plugin Features

### Quick Generate

Select any layer and use `Plugins → PowerAI → Quick Generate` to create an image based on your layer's context.

### Custom Prompts

Use `Plugins → PowerAI → Custom Prompt` to enter detailed prompts for precise control over the output.

### Batch Generation

Select multiple layers and generate images for all of them at once.

### Style Transfer

Apply AI-powered style transfer to existing images and designs.

### Smart Fill

Automatically fill shapes and containers with AI-generated content.

## Keyboard Shortcuts

- `⌘ + ⇧ + G` - Quick Generate
- `⌘ + ⇧ + P` - Custom Prompt
- `⌘ + ⇧ + B` - Batch Generate
- `⌘ + ⇧ + S` - Style Transfer

## Pricing

Sketch plugin is free to download. Image generation uses your PowerAI quota. Pro plans include advanced features like unlimited generations and priority processing.
