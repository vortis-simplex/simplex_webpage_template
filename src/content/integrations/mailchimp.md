## Overview

The Mailchimp integration automates image generation for your email marketing campaigns. Create eye-catching visuals, headers, and promotional graphics directly within your Mailchimp workflows.

## Key Features

- **Email Header Generation**: Create stunning headers for your newsletters
- **Product Showcases**: Generate product images for promotional emails
- **A/B Testing Images**: Automatically create multiple image variations
- **Campaign Automation**: Generate images based on campaign triggers
- **Brand Consistency**: Maintain visual identity across all campaigns

## How It Works

1. **Connect Your Account**: Link your Mailchimp account to PowerAI
2. **Select Audiences**: Choose which email lists to use
3. **Set Up Triggers**: Configure when images should be generated
4. **Create Templates**: Define image templates with your branding
5. **Launch Campaigns**: Send emails with AI-generated visuals

## Use Cases

- **Newsletters**: Generate unique header images for each newsletter
- **Product Launches**: Create promotional images for new products
- **Seasonal Campaigns**: Generate themed imagery for holidays and events
- **Welcome Emails**: Create personalized onboarding visuals
- **Re-engagement Campaigns**: Generate compelling images to win back subscribers

## Setup Instructions

To set up the Mailchimp integration:

1. Navigate to PowerAI Integrations → Mailchimp
2. Click "Connect" and authenticate with Mailchimp
3. Grant permissions for campaign and audience access
4. Configure your image generation preferences
5. Set up automation rules and triggers
6. Test with a sample campaign

## Technical Details

- **API Version**: Mailchimp Marketing API v3.0
- **Authentication**: OAuth 2.0 with API keys
- **Supported Campaigns**: Regular, Plain-text, A/B Test, RSS-driven
- **Image Formats**: JPG, PNG (optimized for email)
- **Max Resolution**: 1200x628 pixels (recommended for email)

## Best Practices

- **Keep it Simple**: Use clear, focused prompts for best results
- **Test First**: Always test images before sending to large audiences
- **Brand Guidelines**: Include brand colors and style in your prompts
- **Mobile Optimization**: Ensure images look good on mobile devices
- **File Size**: Keep images under 100KB for faster email loading
